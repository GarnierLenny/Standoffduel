# Duel audio (optional)

Drop three files here to replace the built-in WebAudio synth cues:

| File          | When it plays                  | Suggested vibe                         |
|---------------|--------------------------------|----------------------------------------|
| `tension.mp3` | Eye-zoom phase (loops)         | Morricone-style ominous drone / whistle |
| `gunshot.mp3` | The DRAW signal                | Single sharp revolver crack            |
| `victory.mp3` | Winner's result screen         | Short triumphant western sting         |

If a file is missing, the game synthesizes that cue automatically - so this is
purely for polish. Keep them small (a few hundred KB) for fast loading.
